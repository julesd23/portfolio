Projects

Real Time Bus Tracker

This is an excercise in using a couple different api's to create a simple bus tracker app. The Mapbox api is used to show a map with styles and icons on the page, and the MBTA api is used to gather all of the location data for the active MBTA busses in Boston.

Bad-Bank

A simulation banking application where one can create an account, log in, deposit, and withdraw funds.

React Todo List

A simple todo list application, but developed with React and modern JSX.

Pokedex App

An exercise in using a simple API to fetch data about the original 151 Pokemon.

PacMan

This is a fun JavaScript demonstration of generating physics animations and cycling through an array of images. While working through this activity I gained a deeper understanding of loops and logic statements.

Mouse Tracker

Instead of using Javascript logic to enact certain physics animations, these CSS animated eyes are entirely dependent on mouse movement instead.
